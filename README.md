# hansincat

学校内の猫を管理し、写真を共有できるアンドロイドアプリです。

開発言語はKotlin
データベースはFirebase

機能
1. 会員登録とログインを実装
2. 投稿の作成と修正と削除の実装
3. コメント機能の実装
4. 写真掲示物検索機能
5. マイページ実装
6. イメージ分析機能
